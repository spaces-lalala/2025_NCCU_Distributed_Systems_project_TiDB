# UI 功能測試檢查清單

## 商品詳情頁面修復
### 已修復的問題：
1. ✅ **ElementPlusError: min should not be greater than max** 
   - 問題：當商品庫存為0時，`el-input-number` 的 `min="1"` 大於 `max="0"`
   - 解決：使用 `Math.max(1, product.stock)` 確保 max 至少為 1
   - 解決：當庫存為 0 時禁用數量選擇器

2. ✅ **數據載入錯誤處理**
   - 加強了 API 響應驗證
   - 添加了詳細的錯誤信息
   - 改善了載入狀態顯示

3. ✅ **庫存狀態視覺化**
   - 添加了庫存不足和售完的視覺指示
   - 動態按鈕文字（售完時顯示"已售完"）
   - 庫存不足時禁用相關功能
